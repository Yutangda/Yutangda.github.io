什么是React

React 是一个用于构建用户界面的 JavaScript 库，能够帮助你操作DOM呈现页面效果，它采用了组件化模式、声明式编码，能提高开发效率与组件复用率，并且在React Native 中可以使用 React语法进行移动端开发

使用了虚拟DOM和优秀的Diffing语法，减少了与真实DOM的交互

React 是由 FaceBook 开发的

jsx 语法规则：

  1. 定义虚拟DOM时，不要写引号

  2. 标签中混入js表达式要用{}

  3. 样式的类名指定不要用class 要用className

  4. 内联样式，要用style={{key:value}}的形式去写

  5. 只有一个根标签 且 标签必须闭合

  6. 标签首字母 若小写开头 则转化为html中的同名元素 若大写开头 则被认定是组件
